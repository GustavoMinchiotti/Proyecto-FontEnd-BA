# Vintage 80’s

Sitio web desarrollado como práctica de maquetación y diseño web, con temática retro inspirada en la cultura pop de las décadas de 1980 y 1990. El proyecto busca aplicar los principios fundamentales de HTML y CSS en la creación de una página estructurada, accesible y visualmente coherente.

---

## Estructura del proyecto

**Archivos principales:**

- `index.html`: documento base que contiene la estructura del sitio.
- `style.css`: hoja de estilos principal que define el diseño visual y la maquetación.
- Carpeta `assets/`: contiene las imágenes y otros recursos utilizados en el sitio.

**Secciones del sitio:**

1. **Hero**: encabezado principal con fondo ilustrativo y texto destacado.
2. **Productos**: muestra de calcos con temática ochentosa, organizada mediante tarjetas.
3. **Reseñas**: opiniones de clientes, maquetadas con CSS Grid.
4. **Contacto**: formulario funcional de envío implementado con Formspree.

---

## Tecnologías utilizadas

- **HTML5**: para la estructura y el contenido del sitio.
- **CSS3**: para el diseño visual, maquetación con Flexbox y Grid, y adaptabilidad.
- **Formspree**: servicio externo para el procesamiento de formularios.
- **Visual Studio Code** con extensión **Prettier**: herramienta de desarrollo y formateo de código.

---

## Diseño y estilo

- Diseño **minimalista y centrado** en la experiencia del usuario.  
- Paleta de colores basada en tonos cálidos, destacando el color principal:
  - Botón principal: `#FF6707`
  - Hover de botón: `#fc4700`
- Uso de **contenedores amplios** y **espaciado uniforme** para lograr una composición visual equilibrada.
- Maquetación basada en **flexbox y grid**, con preparación para adaptabilidad responsiva.

---

## Responsividad

El sitio está optimizado para pantallas de escritorio de hasta **1980 px** de ancho y preparado para ajustes progresivos en versiones móviles.

- Se usaron media queries progresivas `(max-width)` para adaptar el contenido a distintos tamaños de pantalla.
- Se priorizó un diseño mobile-first en ajustes visuales, reduciendo anchos, columnas y tamaños de texto.
- Se mantuvo la simplicidad del CSS y la legibilidad del código, cumpliendo con las pautas académicas.
